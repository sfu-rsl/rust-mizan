# Contributing

There are three main ways to contribute to RustMizan. Each has its own guide.

| Contribution | Guide |
| --- | --- |
| Add a new vulnerability to the dataset | [Add a vulnerability](vulnerabilities.md) |
| Add a new mutation | [Add a mutation](mutations/index.md) |
| Submit evaluation results to the leaderboard | [Submit leaderboard results](leaderboard.md) |

To report a problem (a mislabeled sample, a compile failure, a bug) or ask a question, please [open an issue](https://github.com/sfu-rsl/rust-mizan/issues).
